# Trouve ton artisan

Projet réalisé dans le cadre du devoir CEF.

## Technologies
- ReactJS
- Bootstrap
- Sass
- Node.js
- Express
- Sequelize
- MySQL
- Git / GitHub

## Installation

### 1. Base de données
Créer la base avec les fichiers :
- `database/schema.sql`
- `database/seed.sql`

### 2. Backend
```bash
cd backend
npm install
npm run dev
```

### 3. Frontend
```bash
cd frontend
npm install
npm run dev
```

## URLs
- Frontend : `http://localhost:5173`
- Backend : `http://localhost:3001`

## Fonctionnalités
- Liste des catégories depuis la base
- Recherche par nom d’artisan
- Liste des artisans par catégorie
- Fiche artisan
- Formulaire de contact
- Pages légales
- Page 404
